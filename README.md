

**README — Diversification Internationale \& Construction de Portefeuilles (MATLAB)**



**Projet académique – Théories Avancées de Portefeuille (UQAM, 2025)**



**1. Objectif du projet**

Ce projet vise à analyser la diversification internationale des portefeuilles boursiers pour les périodes 1998–2007 et 2008–2020.



**2. Structure du dépôt**

Code/

&nbsp; - portefeuille\_main.m

&nbsp; - portefeuille\_classement.m



Data/

&nbsp; - gport2.xlsx



Report/

&nbsp; - Diversification-Internationale.pdf



README.md



**3. Scripts MATLAB**

• portefeuille\_main.m  

&nbsp; → Analyse statistique, estimation des paramètres, construction des portefeuilles Alpha, Beta, Delta, Gamma, Segmentés et Intégrés.



• portefeuille\_classement.m  

&nbsp; → Génère les tableaux des pays les plus et les moins segmentés par période.



**4. Exécution**

Dans MATLAB :

addpath('Code'); 

addpath('Data');



portefeuille\_main

portefeuille\_classement



Les fichiers Excel contenant les portefeuilles seront générés automatiquement.



**5. Données**  

Il contient :  

• Rendements MSCI  

• Rendements des pays  

• Capitalisation boursière  

pour les deux périodes étudiées.
